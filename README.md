1. var vs let vs const
=> var is function-scoped and can be re-declared; let & const are block-scoped. let can change value, const can be reassigned after declaration.

2. Spread operator(...)
=> It is used to expand or copy elements of array/objects. Example: [...arr] copies an array.

3. map() vs filter() vs foEach()
=> map() returns a new array after modifying each element; filter() returns elements based on condition. foEach() just a loop.

4. Arrow function
=> A shorter way to write functions using (=>). Example: const sum = (a,b) => a+b;

5. Template literals
=> Use backticks `` to write strings and include variables easily.
